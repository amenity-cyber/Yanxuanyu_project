上传前后端交互网站

1.功能特点

学生选课/退课
课程容量限制
防止重复选课
实时数据统计
响应式界面

2.技术栈

后端：Flask 2.0.3, MySQL
前端：Vue.js 2, Axios
样式：原生CSS
